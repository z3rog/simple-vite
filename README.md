# Simple-Vite

Simple-vite it's a demo serves as an example for my personal blog.It explains the how Vite[https://github.com/vitejs/vite] works.

# Usage

At first:

```js
git clone https://github.com/z3rog/simple-vite.git
cd simple-vite
yarn
yarn build
yarn link
```

then use it in your personal dir:

```js
yarn link simple-vite
simple-vite
```
